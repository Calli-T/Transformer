# 모듈 클래스는 초기화 메서드와 순방향 메서드를 재정의하여 활용
# 초기화는 신경망 계층 초기화, 순방향 메서드는 모델 구조 정의
# 객체 호출하는 순간 순방향 메서드 실행, 실행되면 정의한 순서대로 학습을 진행

초기화 메서드는 super 함수로 모듈 클래스 속성을 초기화.
super 함수로 부모 클래스를 초기화 하면 서브 클래스인 모델에서 부모 클래스의 속성을 사용가능
초기화 하고 학습에 사용되는 계층을 메서드에 선언

순방향 메서드는 초기화 메서드에서 선언한 모델 매개변수를 활용해 신경망 구조를 설계

층은 init에, 값을 전달하는건 forward에서 하는듯

초기화 메서드에서 super 함수로 부모 클래스를 초기화 했으므로
역방향 backward 연산은 정의하지 않아도 된다고한다.
-backpropagation 자동인가? 파이토치 자동미분 autograd에서 모델의 매개변수를 역으로 전파해서 기울기 또는 변화도를 계산해준다고한다.
