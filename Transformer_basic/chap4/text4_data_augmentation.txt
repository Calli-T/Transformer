데이터 증강: 데이터가 가진 고유한 특징을 유지한 채
변형하거나 노이즈를 추가해
데이터셋의 크기를 인위적으로 늘리는 방법

데이터 증강은 모델의 과대적합을 줄이고 일반화 능력을 향상 시킬 수 있다고 한다.
---------------------------------
텍스트 데이터의 증강은 삽입 삭제 교체 대체 생성 반의어 맞춤법 교정 역번역등이 있다고 한다.
NLPAUG 라이브러리
pip install numpy requests nlpaug transformers sacremoses nltk
... 자연어 처리 데이터 증강 라이브러리 설치방법이라고 한다
해당 라이브러리는 넘파이와 리퀘스트츠 라이브러리에 대해 종속,
트랜스포머스는 이전에 학습된 모델을 활용하는 API제공
새크리모시스와 NLTK는 텍스트 토큰화와 정규화
---------------------------------
(텍스트 데이터 증강으로써의) 삽입과 삭제
삽입: 의미 없는 문자나 단어 또는 문장 의미에 영향을 끼치지 않는 수식어 등을 추가하는 방법
삭제: 입의의 단어나 문자를 삭제해 데이터의 특징을 유지하는 방법

nlpaug 라이브러리의 augmenter의 하위 클래스들로 가능함
RandomCharAug와 RandomWordAug는 단위가 문자하나이냐 단어이냐 차이인듯
RandomWordAug는 insert, substitute(대체), swap, delete, crop(자르기) 기능 제공

RandomWordAug의 대체의 경우 본래의 문맥과 전혀 다른 문장으로 생성이 될 수도있음/DB내 유의어나 동의어로 변경하는것
대체의 경우, 모델을 활용하여 대체하는 경우 ContextualWordEmbsAug사용

역번역: 입력 텍스트를 특정 언어로 번역한 다음 다시 본래의 언어로 번역하는것
해당과정에서 패러프리이징 효과를 볼 수도 있다
패러프레이징: 뜻이 같거나 유사한 어휘를 사용해 문장을 바꿔 표현하는 것
